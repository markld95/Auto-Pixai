## 📢 Credits & Modifications
This is an English-localized fork of the original [auto-pixai](https://github.com/Mr-Smilin/auto-pixai) project by **Mr-Smilin**.

**Key Enhancements in this Fork:**
* **Full English Support:** Logs and error messages translated for easier troubleshooting.
* **Cookie Authentication:** Added `PIXAI_COOKIE` support to bypass login screens and Cloudflare issues.
* **Docker Optimized:** Specifically configured to run in Docker without memory crashes.
* **Smart Search Logic:** Improved "Claim" button detection that survives website UI updates.
